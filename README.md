HRA is a repository that defines TigerGraph Graph database schema for capturing a user Health Risk Assessments.

The data to load into the database is defined in the GraphData/ directory. 

The src/ directory contains 3 python programs. CreateUserProfile.py program generates a list of Persons and associated attributes, like gender, birthdays etc. The second program, CreateClinicalProfile.py will generate a clinical profile of a member, with a sample of a memberâ€™s risks.  The third program, app_profile.py file is a Streamlit app that analysis the data generated by the other programs to see how well your data is 'balanced.'
